# Component
Components are small script file that have template and other properties that are loaded when they are called. Component's template part can have `key` that are written as `{key_name}` inside the template. These can be used to manipulate attribute's name, value or the element's inner html. Whenever the key's value is updated, instead of updating whole component, all the elements that have that key will be updated.
