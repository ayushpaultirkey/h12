# Component
### Property
### &bull; `id`

This is used to set the unique ID for the component. For custom ID use it inside `constructor` or when using component tag add `id` key inside the `args` attribute.

*Example:* When using component tag `<Button args={{ id: "button-1" }} />`

### &bull; `child`

An object that store all the child component's in it.

### &bull; `parent`

An object that store the current component's parent.

### &bull; `root`

It store the root element of the component.

### &bull; `args`

It store argument that was passed when the component was created.
